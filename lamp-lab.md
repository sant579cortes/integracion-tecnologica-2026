# Laboratorio Servidor LAMP

1. **¿Qué es Apache**?: Es un servidor web de código abierto que recibe peticiones HTTP y entrega páginas web a los usuarios.
2. **¿Qué es LAMP?**: Es el conjunto de Linux (S.O.), Apache (Servidor), MySQL (Base de datos) y PHP (Lenguaje de programación) para entornos web.
3. **Diferencia Apache vs NGINX**: Apache es altamente modular y flexible, mientras que NGINX destaca por su velocidad y bajo consumo de recursos al manejar muchas conexiones al tiempo.
4. **Capturas de Evidencia**:
   - Localhost (Página personal diseñada).
   - phpinfo (Verificación de PHP funcionando).
   - phpmyadmin (Gestión de base de datos accesible).
